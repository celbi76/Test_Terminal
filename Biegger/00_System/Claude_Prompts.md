# Claude AI – Prompt-Bibliothek

> Diese Sammlung enthält einsatzbereite Prompts für jede Phase der Wissensarbeit. Kopiere den gewünschten Prompt, füge deinen Inhalt ein und starte das Gespräch mit Claude.

**Hinweis zur Verwendung:** Ersetze alle `[IN ECKIGEN KLAMMERN]` durch deinen eigenen Inhalt. Prompts können kombiniert und angepasst werden.

---

## 1. Notizverarbeitung – Roh zu Strukturiert

### 1.1 Inbox-Notiz verarbeiten

```
Ich habe eine rohe Notiz aus meiner Inbox. Bitte hilf mir, sie zu verarbeiten.

ROHE NOTIZ:
[Füge deine Inbox-Notiz hier ein]

Bitte:
1. Identifiziere den Kerngedanken (max. 1–2 Sätze)
2. Schlage vor, ob diese Notiz besser als: (a) Zettel, (b) Projektreferenz, (c) Ressource, oder (d) Archiv passt
3. Extrahiere 3–5 relevante Schlüsselwörter als Tags
4. Formuliere eine klare Titelzeile für diese Notiz
5. Wenn Zettel: Formuliere den Gedanken als eigenständige, atomare Aussage in eigenen Worten
```

### 1.2 Mehrere Notizen gleichzeitig sortieren

```
Ich habe folgende Inbox-Notizen angesammelt. Hilf mir, jede einer PARA-Kategorie zuzuordnen.

PARA-Kategorien:
- Projekte: zeitgebunden, klares Ergebnis, aktiv
- Bereiche: dauerhafte Verantwortung, kein Enddatum
- Ressourcen: thematisches Referenzmaterial, interessiert mich
- Zettelkasten: destillierte Erkenntnis, eigenständiger Gedanke
- Archiv: abgeschlossen oder inaktiv

NOTIZEN:
1. [Notiz 1]
2. [Notiz 2]
3. [Notiz 3]

Für jede Notiz: Kategorie + kurze Begründung (1 Satz).
```

## 2. Zusammenfassung und Destillation

### 2.1 Artikel oder Text zusammenfassen

```
Bitte fasse folgenden Text nach der "Progressive Summarization"-Methode zusammen:

TEXT:
[Text einfügen]

Erstelle:
1. STUFE 1 – Wichtigste Passagen (als Zitate, max. 30% des Originals)
2. STUFE 2 – Fettgedruckte Kernaussagen aus Stufe 1 (max. 15%)
3. STUFE 3 – Executive Summary in eigenen Worten (3–5 Sätze)
4. KERNAUSSAGE – Der eine zentrale Gedanke des Textes (1 Satz)
5. MEINE REAKTION – Was überrascht dich? Was willst du weiterverfolgen?
```

### 2.2 Buch oder langen Text verarbeiten

```
Ich lese gerade [BUCHTITEL] von [AUTOR]. Hier sind meine Lesenotizen:

[Lesenotizen einfügen]

Bitte hilf mir:
1. Die 3–5 wichtigsten Erkenntnisse aus diesen Notizen herauszuarbeiten
2. Jeden dieser Punkte als eigenständigen Zettel-Entwurf zu formulieren (atomar, in eigenen Worten, mit Quellenangabe)
3. Verbindungen zu diesen allgemeinen Themen aufzuzeigen: [deine Interessensgebiete nennen]
4. 2–3 Fragen vorzuschlagen, die ich mir beim Weiterlesen stellen sollte
```

### 2.3 Podcast oder Video-Transkript verarbeiten

```
Ich habe folgende Notizen aus einem Podcast/Video gemacht:

QUELLE: [Name, URL]
NOTIZEN:
[Notizen einfügen]

Bitte:
1. Identifiziere die 3 Hauptthesen oder Argumente
2. Welche Aussagen sind besonders originell oder überraschend?
3. Formuliere 2–3 atomare Zettel-Entwürfe mit dem Format:
   TITEL: [Aussage als vollständiger Satz]
   INHALT: [Erklärung in 2–4 Sätzen]
   QUELLE: [Quellenangabe]
   TAGS: [3–5 Tags]
```

## 3. Verbindungen finden

### 3.1 Verbindungen zu bestehenden Ideen entdecken

```
Ich habe diesen Zettel/Gedanken:

[Zettelinhalt einfügen]

Und diese bestehenden Themen/Bereiche in meinem Zettelkasten:
[Liste deiner Hauptthemen, z.B.: Produktivität, Lernen, Kreativität, Führung]

Bitte:
1. Welche nicht-offensichtlichen Verbindungen siehst du zwischen diesem Gedanken und den genannten Bereichen?
2. Welche Spannungen oder Widersprüche könnten interessant sein?
3. Welche Metaidee verbindet diese scheinbar verschiedenen Dinge?
4. Schlage 3 Fragen vor, die durch diese Verbindungen entstehen
```

### 3.2 Querverbindungen zwischen zwei Themen

```
Ich beschäftige mich mit zwei scheinbar unverbundenen Themen:

THEMA A: [Thema und kurze Beschreibung]
THEMA B: [Thema und kurze Beschreibung]

Bitte zeige mir:
1. Strukturelle Ähnlichkeiten (wie ähneln sich die Muster, auch wenn der Inhalt verschieden ist?)
2. Mögliche Analogien (Was aus Thema A kann als Metapher für Thema B dienen?)
3. Mögliche Synthese: Gibt es eine übergeordnete Idee, die beide verbindet?
4. Konkrete Beispiele, wo beide Bereiche sich in der Praxis überschneiden
```

### 3.3 Lücken im eigenen Denken finden

```
Hier ist mein aktuelles Verständnis von [THEMA]:

[Deine Notizen oder Zusammenfassung einfügen]

Bitte analysiere:
1. Welche logischen Lücken oder Sprünge siehst du in meinem Denken?
2. Was nehme ich als selbstverständlich an, ohne es begründet zu haben?
3. Welche Gegenargumente oder alternativen Perspektiven fehlen?
4. Was müsste ich als nächstes lesen/lernen, um mein Verständnis zu vertiefen?
5. Welche 2–3 Zetteln sollte ich noch schreiben, um dieses Thema zu vervollständigen?
```

## 4. Atomare Zettel erstellen

### 4.1 Zettel aus Rohgedanken formulieren

```
Ich habe diesen rohen Gedanken:

[Rohgedanke einfügen]

Bitte hilf mir, daraus einen guten Zettelkasten-Zettel zu machen:

Ein guter Zettel ist:
- Atomar (ein einziger Gedanke)
- Autonom (verständlich ohne Kontext)
- In eigenen Worten (keine Paraphrase, sondern Transformation)
- Permanent (gültig unabhängig von Zeit und Projekt)

Erstelle:
1. TITEL: Als vollständige Aussage (nicht als Frage oder Thema)
2. KERNAUSSAGE: 1–3 Sätze, die den Gedanken vollständig ausdrücken
3. ERLÄUTERUNG: 2–4 Sätze Kontext oder Begründung
4. MÖGLICHE LINKS: Welche Themen könnte dieser Zettel verbinden?
5. TAGS: 3–5 passende Schlagworte
```

### 4.2 Zettel kritisch überprüfen

```
Bitte überprüfe diesen Zettel aus meinem Zettelkasten:

[Zettelinhalt einfügen]

Checkliste:
1. Ist dieser Zettel wirklich atomar, oder enthält er mehrere Gedanken? Falls mehrere: Wie würdest du ihn aufteilen?
2. Ist der Titel eine vollständige Aussage (nicht nur ein Thema)?
3. Ist der Zettel ohne Kontext verständlich?
4. Gibt es Formulierungen, die unklar oder zu abstrakt sind?
5. Was fehlt, damit dieser Gedanke vollständig ausgedrückt ist?
6. Bitte formuliere eine verbesserte Version.
```

### 4.3 Literaturnotiz zu permanenter Notiz transformieren

```
Ich habe diese Literaturnotiz (was ich aus einer Quelle gelernt habe):

QUELLE: [Quellenangabe]
NOTIZ: [Literaturnotiz einfügen]

Bitte hilf mir, 2–3 permanente Zettel daraus zu destillieren.

Für jeden Zettel:
- Die Aussage muss von mir stammen, nicht vom Autor
- Sie muss über die spezifische Quelle hinaus gültig sein
- Sie soll anschlussfähig an andere Ideen sein

Formuliere jeden Zettel vollständig mit Titel, Kernaussage und Tags.
```

## 5. Schreiben verbessern

### 5.1 Text strukturieren

```
Ich möchte über folgendes Thema schreiben:

THEMA: [Thema]
ZIELGRUPPE: [Wer liest das?]
ZIEL: [Was soll der Leser danach wissen/fühlen/tun?]

Meine bisherigen Gedanken/Notizen:
[Notizen einfügen]

Bitte hilf mir:
1. Eine klare Hauptthese zu formulieren (1 Satz)
2. Eine logische Struktur mit 3–5 Hauptpunkten vorzuschlagen
3. Die stärksten und überzeugendsten Argumente zu identifizieren
4. Den besten Einstieg zu skizzieren (erste Sätze, die Neugier wecken)
5. Potenzielle Einwände des Lesers vorauszudenken
```

### 5.2 Entwurf verbessern

```
Hier ist mein Entwurf:

[Text einfügen]

Bitte gib mir Feedback zu:
1. KLARHEIT: Wo ist die Aussage unklar oder missverständlich?
2. STRUKTUR: Ist der logische Aufbau nachvollziehbar? Was fehlt oder ist überflüssig?
3. FORMULIERUNG: Welche Sätze sind zu lang, zu passiv oder zu abstrakt?
4. ÜBERZEUGUNGSKRAFT: Was würde ein kritischer Leser in Frage stellen?
5. STÄRKEN: Was funktioniert besonders gut und soll beibehalten werden?

Formuliere für jeden Punkt konkrete Verbesserungsvorschläge.
```

### 5.3 Komplex zu einfach

```
Ich habe diesen komplexen Sachverhalt verstanden und möchte ihn einfach erklären:

[Komplexer Inhalt oder eigene Erklärung einfügen]

ZIELGRUPPE: [Alter/Vorkenntnisse der Zielgruppe]

Bitte:
1. Erkläre den Kern mit einer guten Analogie oder einem konkreten Beispiel
2. Verwende max. 150 Wörter für die Erklärung
3. Vermeide Fachbegriffe (oder erkläre sie in einem Satz)
4. Prüfe: Versteht jemand ohne Vorkenntnisse die Essenz?
```

## 6. Wöchentliche Review

### 6.1 Wochenreview durchführen

```
Wochenreview für KW [NUMMER], [DATUM VON] bis [DATUM BIS]

Meine Wochennotizen:
[Tagesnotizen oder Zusammenfassung der Woche einfügen]

Bitte hilf mir bei der Reflexion:

1. LEISTUNGEN: Was habe ich diese Woche wirklich geschafft? (nicht nur was war geplant)
2. ERKENNTNISSE: Was habe ich gelernt – über Themen, über mich, über meine Arbeit?
3. MUSTER: Gibt es wiederkehrende Themen, Probleme oder Energiegeber?
4. ZETTEL-KANDIDATEN: Welche Gedanken dieser Woche verdienen einen permanenten Zettel?
5. NÄCHSTE WOCHE: Welche 3 Dinge hätten den grössten Hebel?
6. DANKBARKEIT: Was lief besser als erwartet?

Halte die Antworten konkret und handlungsorientiert.
```

### 6.2 Monatliche Rückschau

```
Monatsrückschau für [MONAT] [JAHR]

Meine Notizen und Projekte des Monats:
[Zusammenfassung einfügen]

Fragen für die Rückschau:

1. PROJEKTE: Welche Projekte habe ich abgeschlossen, gestartet, pausiert?
2. BEREICHE: Welche Lebensbereiche haben Aufmerksamkeit bekommen, welche zu wenig?
3. LERNEN: Welches Wissen hat sich diesen Monat am meisten verdichtet?
4. GEWOHNHEITEN: Was hat funktioniert, was nicht?
5. QUALITÄT DES DENKENS: Gibt es Themen, bei denen ich in Kreisen gedacht habe statt vorwärts?
6. ZETTELKASTEN-GESUNDHEIT: Welche Bereiche des Zettelkastens fehlen oder sind dünn?
7. INTENTION FÜR NÄCHSTEN MONAT: Ein Satz, der den Fokus des nächsten Monats beschreibt.
```

## 7. Projektarbeit

### 7.1 Projekt kickstarten

```
Ich starte ein neues Projekt:

PROJEKTTITEL: [Titel]
ZIEL: [Was soll am Ende stehen?]
FRIST: [Deadline]
KONTEXT: [Was ich bereits weiss / bisherige Gedanken]

Bitte hilf mir:
1. Das Ziel schärfer zu formulieren (SMART-Kriterien: spezifisch, messbar, attraktiv, realistisch, terminiert)
2. Die 5–7 wichtigsten Meilensteine zu identifizieren
3. Mögliche Hindernisse und Risiken vorauszudenken
4. Die erste konkrete Aktion zu benennen (unter 30 Minuten)
5. Relevante Ressourcen oder Fragen zu identifizieren, die ich klären muss
```

### 7.2 Projekt-Blockade lösen

```
Ich stecke bei einem Projekt fest:

PROJEKT: [Projektname und Ziel]
WO ICH STECKE: [Beschreibe die Blockade]
WAS ICH SCHON VERSUCHT HABE: [Bisherige Ansätze]
WAS MICH HEMMT (vermutlich): [Deine Einschätzung]

Bitte:
1. Stelle mir 3 klärende Fragen, die ich mir selbst beantworten sollte
2. Identifiziere, ob die Blockade kognitiv (unklar was zu tun ist), emotional (Angst, Widerstand) oder praktisch (fehlende Ressourcen) ist
3. Schlage 3 verschiedene Wege vor, die Blockade zu umgehen oder aufzulösen
4. Was wäre der kleinstmögliche nächste Schritt, um wieder in Bewegung zu kommen?
```

## 8. Ideen entwickeln

### 8.1 Idee durchdenken

```
Ich habe eine Idee:

[Idee beschreiben]

Bitte denk mit mir:
1. Was ist das Kernversprechen dieser Idee? (Für wen, welcher Nutzen?)
2. Welche Annahmen stecken darin, die ich überprüfen sollte?
3. Stärkste Argumente FÜR diese Idee
4. Stärkste Argumente GEGEN diese Idee
5. Welche Variante dieser Idee wäre noch interessanter oder origineller?
6. Was wäre ein erster konkreter Test, ob die Idee funktioniert?
```

### 8.2 Brainstorming anschieben

```
Ich brauche Ideen für:

KONTEXT: [Worum geht es?]
ZIEL: [Was soll mit den Ideen erreicht werden?]
CONSTRAINTS: [Einschränkungen, z.B. Zeit, Budget, Fähigkeiten]

Bitte generiere:
1. 5 naheliegende, konventionelle Ideen
2. 5 unkonventionelle, überraschende Ideen
3. 2 "Verrückte" Ideen, die eigentlich unmöglich klingen
4. Die 1 Idee, die du persönlich am interessantesten findest – und warum
```

## 9. Denken schärfen

### 9.1 Annahmen sichtbar machen

```
Hier ist eine Überzeugung oder ein Argument von mir:

[Deine Überzeugung oder dein Argument einfügen]

Bitte dekonstruiere es:
1. Welche versteckten Annahmen stecken darin?
2. Unter welchen Bedingungen wäre das Gegenteil wahr?
3. Welche Denkfehler (kognitive Verzerrungen) könnten hier am Werk sein?
4. Wie würde jemand argumentieren, der das Gegenteil vertritt?
5. Wie kann ich diese Überzeugung testen – was würde sie falsifizieren?
```

### 9.2 Sokratisches Gespräch

```
Ich möchte mein Denken zu einem Thema schärfen durch sokratisches Fragen.

THEMA/ÜBERZEUGUNG: [Dein Standpunkt]

Bitte führe mich durch ein sokratisches Gespräch:
- Stelle mir eine Frage auf einmal
- Warte auf meine Antwort bevor du die nächste stellst
- Ziel ist nicht, mich zu widerlegen, sondern mein Denken zu präzisieren
- Beende das Gespräch nach 5–7 Fragen mit einer kurzen Zusammenfassung

Beginne mit der ersten Frage.
```

## 10. Tagesnotiz-Auswertung

### 10.1 Erkenntnisse aus Tagesnotizen destillieren

```
Hier sind meine Tagesnotizen der letzten Woche:

[Tagesnotizen einfügen]

Bitte extrahiere:
1. WIEDERKEHRENDE GEDANKEN: Welche Themen oder Sorgen tauchen mehrmals auf?
2. ENERGIEGEBER: Wann war die Energie/Stimmung besonders gut? Was war der Auslöser?
3. ENERGIENEHMER: Wann war die Energie/Stimmung besonders tief?
4. ZETTEL-KANDIDATEN: Welche Gedanken verdienen eine permanente Notiz?
5. OFFENE FRAGEN: Welche Fragen sind diese Woche aufgetaucht und noch ungelöst?
6. MUSTER: Gibt es Verhaltens- oder Denkmuster, die dir auffallen?
```

## 11. Schnellprompts (für täglich)

### Schnell-Zettel
```
Formuliere diesen Gedanken als atomaren Zettel (Titel als vollständige These + 3–5 Sätze Erklärung + 3 Tags):

[Gedanke]
```

### Schnell-Zusammenfassung
```
Fasse das in 3 Kernaussagen zusammen. Jede Aussage in max. 1 Satz:

[Text]
```

### Schnell-Tags
```
Schlage 5 passende Obsidian-Tags für diesen Inhalt vor (ohne #, klein geschrieben):

[Inhalt]
```

### Schnell-Titel
```
Formuliere 3 mögliche Titel für diese Notiz. Der Titel soll eine vollständige These sein, kein Thema:

[Notizinhalt]
```
